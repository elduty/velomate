Never add `Co-Authored-By: Claude` or any AI-related attribution in commit messages.

**Why:** User is publishing open-source and doesn't want the code to appear AI-generated.

**How to apply:** Commit messages should be clean, human-style. No co-author lines, no "Generated with Claude Code", no AI mentions anywhere in commits or PR descriptions.
