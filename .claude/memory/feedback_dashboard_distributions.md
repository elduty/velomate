Don't remove HR Distribution or Power Distribution panels from the Activity Details dashboard. Zones (5-6 standard buckets) show time-in-zone. Distribution (full histogram) shows the granular shape of the effort. Both are useful and the user explicitly wants both.

**Why:** I incorrectly removed them as "redundant" and the user pushed back — they serve different analytical purposes.

**How to apply:** When decluttering dashboards, don't remove panels that show the same data at different granularities. Ask first if unsure whether two views are redundant.
